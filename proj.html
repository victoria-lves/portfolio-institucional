Sobre o Projeto

Este projeto visa desenvolver uma solução web integrada para a gestão acadêmica de escolas públicas. O sistema permite o controle de frequência, lançamento de notas e comunicação direta entre pais e mestres. A plataforma está sendo construída com tecnologias modernas para garantir acessibilidade e rapidez, focando na experiência do usuário final e na redução da burocracia escolar.

Objetivos
  • Automatizar diários de classe.
  • Reduzir uso de papel em 40%.
  • Implementar app mobile.
Resultados

Até o momento, o protótipo da interface foi validado com 5 diretores. O banco de dados foi modelado e a API RESTful está 60% concluída.
